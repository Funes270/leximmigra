# LexImmigra

Gestionale Legale Completo per il Diritto dell'Immigrazione

## Stack Tecnologico

- **Frontend:** React 19 + TypeScript + Tailwind CSS + shadcn/ui
- **Backend:** tRPC + Drizzle ORM + Hono + MySQL
- **Autenticazione:** OAuth 2.0

## Moduli

- Dashboard con KPI e statistiche
- Anagrafiche Clienti (cittadini stranieri)
- Pratiche (gestione completa procedure)
- Permessi di Soggiorno (tutte le tipologie)
- Protezione Internazionale (asilo, sussidiaria, umanitaria)
- Contenzioso (ricorsi TAR, appelli, procedure giudiziarie)
- Scadenze e Appuntamenti

## Installazione

```bash
npm install
npm run db:push
npm run dev
```
